## 🚀 Installation

### Option 1: From VS Code Marketplace
1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for `AI Dev Essentials`
4. Click Install

### Option 2: Via Command Line
```bash
code --install-extension ravishan16.ai-dev-essentials
```
